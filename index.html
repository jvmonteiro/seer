Seer

"Seer" is a charting library prototype for JavaScript. It's built using D3.js and currently features bar, line and scatterplot charts.

It also provides support for using CSV and JSON files as data input.

  1. Use the d3.csv(path) or d3.json(path) to load your data.
  2. Create a new chart using a config object passing the desired configuration parameters, mount element and data.
  3. Format the data passing the format object to the chart.format(config) function, according to the desired chart.
  4. Call the chart.render() method to draw the graph on the screen!
See the index.js file for examples.

You can click on the cards to update the data. Have fun :D
